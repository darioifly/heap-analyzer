# F3.S04 — Report: Potree 3D Viewer

## 3D Library Decision

**Chosen**: potree-core@2.0.15 + three@0.184.0
**Decision doc**: docs/decisions/F3.S04-3d-library.md

**Reasoning**:
- potree-core is npm-native ESM, supports Potree 2.0 format (loading2/ directory), integrates with standard Three.js Scene/Renderer/Camera.
- Flexible peer dependency (three >0.125.0) allows latest Three.js.
- Built-in PointColorType enum: RGB, HEIGHT/ELEVATION, CLASSIFICATION — covers all 3 color modes.
- Recently maintained (last npm publish 2026-04-08).

**Rejected**:
- @pnext/three-loader@1.0.0 — pins three ~0.160.0 (outdated), version 1.0.0 suggests stale development.
- Potree viewer (embedded) — not npm-friendly, fights React DOM lifecycle.

## Features Implemented

### Color Modes
- **RGB originale**: original point colors from LAS (PointColorType.RGB)
- **Altezza nDSM**: height gradient blue→red (PointColorType.HEIGHT with heightMin/heightMax)
- **Cumuli**: RGB with highlighted heap overlay meshes

### Camera Presets
- **Dall'alto (Top)**: bird's-eye Z-down view
- **Laterale (Side)**: horizontal from south
- **Reset (Orbit)**: default isometric orbit
- **Centra sulla selezione**: zoom to selected heap bbox

### Base Plane
- Semi-transparent wireframe grid at Z = survey.baseElevation
- Color: evlos-500 (#6575A0), opacity 0.15
- Toggleable via toolbar

### Heap Overlay 3D
- Extruded polygon prisms colored by material category
- Selected heap: higher opacity + white wireframe outline
- Toggleable via toolbar
- Click picking: raycaster against overlay meshes → sets selectedHeapId

### Selection Sync
- 2D → 3D: selectedHeapId from heapStore → heap overlay highlights
- 3D → 2D: click on heap overlay → selectHeap() → synced back to 2D

### Performance Controls
- Point budget slider: 200K → 5M (default 2M)
- Adaptive point sizing (PointSizeType.ADAPTIVE)
- Pixel ratio capped at 2x

### UI Integration
- 2D/3D toggle in HeaderBar (Map/Box icons)
- 3D disabled when Potree data unavailable (tooltip explains)
- Auto-reset to 2D if Potree becomes unavailable
- Loading overlay: spinner + "Caricamento nuvola di punti…"
- Error overlay: EVLOS danger style
- Status bar: point count in Italian locale format

### Lifecycle / Memory
- Full Three.js disposal on unmount (renderer, geometries, materials)
- Point cloud disposed via PointCloudOctree.dispose()
- Heap overlays disposed via disposeGroup()
- ResizeObserver for responsive canvas
